Welcome
Aufgabenstellung
  • Es ist eine Website mit CSS und HTML erstellt.
  • Die Website kann als Präsentation genutzt werden und enthält die Punkte:
    • Einleitung
    • Aufgabenstellung
    • Sollzustand
    • Umsetzung
    • Fazit
  • Die Seite sollen mithilfe der Technik Paralaxes scrolling erreichbar.
  • Über ein Menü sollen die Seiten erreichbar sein.
  • Es sollen VOR und ZURÜCK Buttons vorhanden sein.

Umsetzung
  1. Es ist eine Website mit CSS und HTML erstellt.
  2. Die Website kann als Präsentation genutzt werden.
  3. Die Website enhält Paralaxes scrolling.
  4. Ein Menu ist Vorhanden.
  5. Navigation über vor und zurück Buttons ist möglich.

Schwierigkeiten:

  1. Einarbeitung ins Framework.
  2. Einhaltung von css definitionen innerhalb der HTML-Struktur.
Ergebnis
  1. Paralaxes scrolling scheint eine gute und vervendbare darstellung von Informationen in Websiten zu sein.
  2. Allerdings sollte man für Websiten die kein javascript beinhalten, eine Alternative finden.
  3. Ebenfalls sollte man darauf achten, dass die Informationen auch bei deaktivierten javascript einsehbar sind.
